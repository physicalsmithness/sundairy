IB Economics · Interactive diagrams

The whole set

Every diagram in the IB Economics syllabus, built to be interactive. Drag curves, slide things, watch the dynamics. Change the setting from a St Andrews greengrocer to Albanian wheat to see the same logic survive.

Diagrams are grouped by the canvas they share — the underlying plot type and axes. A student who has learned one diagram on a canvas will find the others immediately readable.
